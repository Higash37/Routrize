# MVP画面仕様

## 画面一覧

| URL | 画面名 | ユーザー |
|---|---|---|
| `/` | Route Builder（ゲスト） | 誰でも |
| `/app/dashboard` | Dashboard | 講師 |
| `/app/routes` | Routes一覧 | 講師 |
| `/app/routes/:id` | Route Builder（ログイン版） | 講師 |
| `/app/students` | 生徒一覧 | 講師 |
| `/app/students/:id` | 生徒詳細 | 講師 |
| `/s` | 生徒ログイン | 生徒 |
| `/s/log` | 学習ログ入力 | 生徒 |

---

## 1. `/` Route Builder（ゲスト）

**目的**: 最短でルート作成体験

### UI構成

- **ヘッダー**
  - ルート名（未保存なら "無題のルート"）
  - [PDF]（A4、TRIAL透かし）
  - [保存]（押下でログイン誘導）
  - [ログイン]

- **本体：ガント**
  - 上: 時間軸（週表示 W1〜W12 をデフォルト）
  - 左固定: 教材行（表紙サムネ + 書名）
  - 右: バー（開始週/週数で描画）

- **追加**
  - `+参考書追加`（モーダル検索）

- **吹き出し（MVP）**
  - 参考書行を選択 → 右サイドパネルで注意/ターニングを追加（各教材最大2つ）

## 2. `/app/dashboard` Dashboard

**目的**: 今日の運用が回る

### 表示内容

- 未入力生徒（最終入力日）
- 今週学習時間（校舎平均または合計）
- 進捗遅れ（MVPは「学習時間が少ない」など簡易判定）

### 導線

- Students へ
- Routes へ
- 直近ルート編集へ

## 3. `/app/routes` Routes一覧

- ルート一覧（校舎共有/自分のみフィルタ）
- 複製
- 新規作成（空/テンプレ）

## 4. `/app/routes/:id` Route Builder（ログイン版）

ゲスト版 + 以下解放:

- 自動保存（または手動保存 + トースト）
- A4/A3 PDF（透かしなし）
- ルートをテンプレ化（校舎共有フラグ）
- 生徒への割当（Students選択）

## 5. `/app/students` 生徒一覧

テーブル形式:

- 生徒コード
- 学年/学校/志望校/偏差値（任意項目）
- 今週学習時間
- 最終入力日
- 未入力日数

クリックで詳細（右ドロワー or 遷移）

## 6. `/app/students/:id` 生徒詳細

- 基本情報（学校/偏差値等）
- 割当ルート（名前 + ガント簡易 or 教材リスト）
- 学習ログ（リスト）
- PIN再発行

## 7. `/s` 生徒ポータル

- ログイン（店舗コード + 生徒ID + PIN）
- 今日の入力
  - 勉強時間（15分刻み）
  - 教材（割当ルートの教材から選択）
  - 理解度
  - 任意メモ
- 過去ログ（リスト）

---

## PDF出力仕様

| 種別 | サイズ | 内容 | 透かし |
|---|---|---|---|
| ゲスト | A4 | チェックリスト（教材名、期間、チェック欄、吹き出し簡易） | `TRIAL` |
| ログイン | A4 | チェックリスト（透かしなし） | なし |
| ログイン | A3 | ロードマップ（左: 表紙+教材名、右: ガント、吹き出し、QR） | なし |

---

## パフォーマンス要件

- 初回 `/` は Route Builder だけ。重いダッシュボードデータは読まない
- `books` はキャッシュ（クライアントキャッシュ or edgeキャッシュ）
- Students一覧はページング
- ルート編集は optimistic update（保存待ちでUIを止めない）
- 画像は遅延読み込み（表紙は低解像から）

---

## 開発マイルストーン

### MS1: ゲストでルート作成（最重要）

- `/` Route Builder（追加・ガント表示）
- LocalStorage保存（ゲスト）
- A4 PDF（TRIAL）

### MS2: ログインとクラウド保存

- Auth（講師）
- `/app/routes` 保存/一覧
- Route Builderの自動保存

### MS3: 生徒運用

- Students CRUD + PIN
- `/s` 生徒ログイン + 日次ログ
- Dashboardの未入力/今週集計

### MS4: 商談用の見栄え

- A3 PDF（表紙+ガント+吹き出し+QR）
- Insights（入力率、時間帯ヒートマップは "可視化のみ"）
